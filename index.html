02 ‚Äî Assembly

The 8-Step
Build Process

Follow this sequence every time. Order matters ‚Äî skipping steps is the #1 cause of failed first boots.

01
Prep
Prepare Workspace & Tools
Clear a large static-safe surface. Lay out all components still in anti-static bags. Gather: Phillips #2 screwdriver, thermal paste, anti-static strap, and zip ties. Clip the strap to an unpainted metal surface before touching anything. Good lighting is non-negotiable.
Pro Tip: Build on a wooden table, never carpet. Carpet generates static electricity that invisibly kills components.
02
CPU
Install CPU onto Motherboard
Open the CPU socket latch. Align the gold triangle on the CPU corner with the triangle on the socket. Lower it gently ‚Äî it drops by gravity alone. Never press or force. Intel: lower the ILM bracket. AMD: no extra latch. Inspect for bent pins before proceeding.
Pro Tip: Bent pins mean a dead board or CPU. If it doesn't seat instantly, lift and realign ‚Äî never force it.
03
RAM
Seat RAM in Dual-Channel Slots
Check your manual for the correct dual-channel slots ‚Äî usually A2 and B2, not A1 and A2. Open both retention clips, align the RAM notch with the slot key, and press firmly and evenly until both clips snap shut. RAM only seats in one direction ‚Äî the notch prevents reversal.
Pro Tip: Enable XMP (Intel) or EXPO (AMD) in BIOS after build to unlock your RAM's advertised speed.
04
SSD
Install M.2 NVMe SSD
Remove the M.2 heatsink if present. Insert the SSD at a 30¬∞ angle into the slot and slide forward until it clicks. Press down flat and fasten the retaining screw. Replace the heatsink. For multiple drives, fill the primary slot (closest to CPU) first for maximum bandwidth.
Pro Tip: Some M.2 slots share bandwidth with SATA ports ‚Äî check your manual to avoid accidentally disabling drives.
05
Cooler
Apply Thermal Paste & Mount Cooler
Apply one pea-sized drop of paste to the center of the CPU heat spreader. Do not pre-spread ‚Äî it distributes under mounting pressure. For AIO coolers, mount the pump head first, route tubing, then mount the radiator. Tighten cooler screws in a cross-pattern to apply even pressure.
Pro Tip: More thermal paste is never better. A pea-sized dot spreads perfectly. Excess causes overflow and shorting.
06
Case
Mount Motherboard in Case
Install brass standoffs matching your board size (ATX, mATX, ITX). Snap the I/O shield into the rear cutout. Lower the board in, align ports with the shield, and start all standoff screws loosely. Tighten them in a cross-pattern. Every standoff hole must have a screw ‚Äî missing ones cause flex and damage.
Pro Tip: Misaligned standoffs will short your motherboard instantly. Verify all hole positions before powering on.
07
Power + GPU
Install PSU, GPU & Route Cables
Mount the PSU fan-down in most modern cases. Install the GPU into the primary PCIe x16 slot until it clicks. Route all cables behind the tray before plugging in: 24-pin ATX, 8-pin CPU EPS, PCIe power to GPU, SATA to drives, fan headers to board. Use velcro ties for management ‚Äî cleaner and reusable.
Pro Tip: Don't close the case yet. Test with side panel off first to catch issues without disassembly.
08
OS Boot
First Boot, BIOS Setup & OS Install
Connect monitor, keyboard, power. Press the power button ‚Äî if fans spin and BIOS loads, you've succeeded. Enter BIOS (usually DEL or F2) and enable XMP/EXPO, verify boot order, check temperatures. Create a Windows 11 or Linux USB on another machine, boot from it, and install. Run chipset and GPU drivers immediately after.
Pro Tip: No display on first boot? Reseat RAM first. 90% of first-boot failures are caused by loose RAM sticks.
